I'm a student engineer

-- Aaron Zhang


This site is primarily used as my student-engineer portfolio. I designed this site to best present myself as a student engineer to professors, project teammates and potential employers. This site is also used for documenting my personal development (projects done, skills gained etc.) along the journey of studying engineering.

To practice front-end developing, this site was coded up from scratch using HTML, CSS, and JavaScript (including Bootstrap and jQuery frameworks). This site was also designed to be "responsive" whose format will change automatically to be better displayed on different-size screens.
